# EMSS Web 架构概览

## 项目简介

Energy-Management-System-Web (EMSS Web) 是一个基于 Vue 3 + TypeScript 的现代化前端管理后台框架，提供企业级能源管理系统的前端支持。

## 核心特性

1. **完整的 RBAC 权限控制系统**
   - 基于角色的权限管理
   - 支持菜单、按钮级别的权限控制
   - 动态路由加载

2. **动态路由模式**
   - 支持静态路由和动态路由两种模式
   - 后端返回用户权限路由
   - 自动注册路由和菜单

3. **组件化架构设计**
   - 通用组件可复用
   - 业务组件独立封装
   - 高级组件组合使用

4. **UnoCSS 原子化样式**
   - 按需生成 CSS
   - 支持自定义预设
   - 与 Element Plus 风格统一

5. **TypeScript 全覆盖**
   - 类型安全的 API 调用
   - 完善的类型定义
   - 组件 Props 类型推导

## 依赖方向

```
页面组件 → 公共组件
    ↓
Hooks → Pinia Store
    ↓
API Service
    ↓
Utils / Axios
```

**严格单向依赖**：
- 上层可以调用下层
- 下层不能反向依赖上层
- 同层之间通过组合复用

---

## 核心设计模式

### 1. Composition API 模式
- 所有组件使用 `<script setup>` 语法
- 逻辑复用通过组合式函数实现
- 类型安全的 Props/Emits 定义

### 2. Setup Store 模式
- Pinia Store 使用 Setup 语法
- State/Getters/Actions 使用 ref/computed/函数
- 避免 Options Store 语法

### 3. 模块化 API 封装
- 按业务模块封装 API
- 统一的请求/响应拦截
- 类型安全的接口调用
