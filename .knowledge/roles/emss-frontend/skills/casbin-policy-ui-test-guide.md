# Casbin 策略管理 UI 功能测试指南

## 测试环境准备

### 前置条件
1. 后端服务已启动（端口 8888）
2. 前端服务已启动（端口 5173 或构建后的端口）
3. 数据库已初始化
4. 至少有一个超级管理员账户（ID=1）

### 测试账户
- 超级管理员：admin / admin123
- 普通用户：user / user123

**步骤：**
1. 访问策略管理页面

**预期结果：**

#### TC-UI-003: 策略搜索筛选

### 2. 创建策略

#### TC-UI-004: 创建基本策略
**步骤：**
1. 点击"新增策略"按钮
2. 填写表单：
   - 角色：选择角色 3
   - 资源路径：/api/v1/test/resource
   - 操作：GET
   - 效果：allow
3. 点击"确定"保存

**预期结果：**
- 策略创建成功
- 提示消息"创建成功"
- 列表中显示新策略

#### TC-UI-005: 创建带上下文的策略
**步骤：**
1. 点击"新增策略"按钮
2. 填写表单：
   - 角色：选择角色 4
   - 资源路径：/api/v1/energy/data
   - 操作：GET
   - 效果：allow
   - 上下文（JSON）：
     ```json
     {
       "factoryIds": [1, 2],
       "shiftScope": "all"
     }
     ```
3. 点击"确定"保存

**预期结果：**
- 策略创建成功
- JSON 格式验证通过
- 列表中显示带上下文标记的策略

#### TC-UI-006: 创建策略表单验证
**步骤：**
1. 点击"新增策略"按钮
2. 不填写任何字段，直接点击"确定"

**预期结果：**
- 表单验证失败
- 显示必填字段错误提示
- 不发送创建请求

#### TC-UI-007: JSON 上下文格式验证
**步骤：**
1. 点击"新增策略"按钮
2. 填写上下文为无效 JSON：`{invalid}`
3. 点击"确定"

**预期结果：**
- 显示 JSON 格式错误提示
- 不发送创建请求

## 权限配置场景测试

### TC-UI-015: 配置工厂管理员权限
**步骤：**
1. 创建策略：
   ```
   角色: 4 (工厂管理员)
   资源: /api/v1/energy/*
   操作: *
   效果: allow
   上下文: {"factoryIds":[1],"shiftScope":"all"}
   ```
2. 验证权限生效

**预期结果：**
- 策略创建成功
- 工厂管理员可访问工厂 1 的能源数据
- 工厂管理员无法访问其他工厂的数据

### TC-UI-016: 配置班组长权限
**步骤：**
1. 创建策略：
   ```
   角色: 5 (班组长)
   资源: /api/v1/energy/data
   操作: GET
   效果: allow
   上下文: {"factoryIds":[1],"shiftScope":"own"}
   ```
2. 验证权限生效

**预期结果：**
- 班组长只能查看本班次数据
- 无法修改数据

## 自动化测试建议

### E2E 测试框架
推荐使用 Playwright 或 Cypress 进行自动化 UI 测试。

### 测试脚本示例
```typescript
// tests/e2e/policy.spec.ts
import { test, expect } from '@playwright/test';

test.describe('策略管理', () => {
  test.beforeEach(async ({ page }) => {
    await page.goto('/login');
    await page.fill('[name="username"]', 'admin');
    await page.fill('[name="password"]', 'admin123');
    await page.click('[type="submit"]');
    await page.waitForURL('/dashboard');
  });

  test('访问策略管理页面', async ({ page }) => {
    await page.goto('/manage/policy');
    await expect(page.locator('h1')).toContainText('策略管理');
  });

  test('创建策略', async ({ page }) => {
    await page.goto('/manage/policy');
    await page.click('text=新增策略');

    // 填写表单
    await page.selectOption('[name="role"]', '3');
    await page.fill('[name="resource"]', '/api/v1/test');
    await page.selectOption('[name="action"]', 'GET');
    await page.click('text=确定');

    // 验证成功
    await expect(page.locator('.el-message--success')).toBeVisible();
  });
});
```
